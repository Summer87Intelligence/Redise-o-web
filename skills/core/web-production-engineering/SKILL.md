# web-production-engineering

## Objetivo
Desarrollo web producción (Next.js, APIs, seguridad, performance)

## Procedimiento

1. Detectar stack
2. Separar server/client
3. Validar endpoints
4. Optimizar queries
5. Entregar patch mínimo

## Reglas
- No romper lo existente
- Validar inputs
- Seguridad server-side
- Cambios mínimos

## Checklist
- ¿Es seguro?
- ¿Escala?
- ¿Mantiene performance?
- ¿Es mantenible?
