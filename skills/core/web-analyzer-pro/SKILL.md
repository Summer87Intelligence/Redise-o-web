# Web Analyzer Pro

## Objetivo
Analizar una URL pública y devolver señales accionables de estructura y contenido.

## Procedimiento
1. Validar URL
2. Obtener HTML con script
3. Extraer title, h1/h2, texto
4. Analizar UX/copy
5. Proponer mejoras

## Reglas
- No inventar contenido
- No acceder a contenido privado
- Informar errores

## Checklist
- ¿URL válida y pública?
- ¿Extracción consistente?
- ¿Salida JSON limpia?
- ¿Sin ejecución remota?
